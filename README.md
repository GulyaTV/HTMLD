# HTMLD
## Теги
1. ```<!DOCTYPE htmld>```: Обязательный тег для объявления типа документа.
2. ```<htmld>```: Корневой элемент документа.
3. ```<head>```: Содержит метаинформацию о странице.
4. ```<title>```: Заголовок страницы.
5. ```<body>```: Основное содержимое страницы.
6. ```<header>```: Шапка страницы.
7. ```<nav>```: Навигационное меню.
8. ```<main>```: Основной контент страницы.
9. ```<footer>```: Подвал страницы.
10. ```<h1>```-```<h6>```: Заголовки.
11. ```<p>```: Параграф.
12. ```<img>```: Изображение.
13. ```<a>```: Ссылка.
## Новые теги (Тег в <>, чтобы закрыть </>):
### цифра.5 - пример
1. ```<dog>```: Тег для вставки изображения или анимации собаки (вдохновлено Гуля ТВ).
    - ```<dog src="dog.gif" alt="Собака Гуля">```
2. ```<woof>```: Тег для добавления звука собачьего лая (автовоспроизведение или по клику).
    - ```<woof src="bark.mp3" autoplay>```
3. ```<paw>```: Тег для создания кнопки в форме собачьей лапы.
    - ```<paw onclick="alert('Гав!')">Нажми на меня!</paw>```
4. ```<tail>```: Тег для создания анимации "виляния хвостом" (CSS-анимация).
    - ```<tail speed="2">Виляние хвостом</tail>```
## Пример кода
```
<!DOCTYPE htmld>
<htmld>
<head>
    <title>Моя страница</title>
</head>
<body>
    <header>Добро пожаловать на сайт Гуля ТВ!</header>
    <nav>
        <a href="/">Главная</a>
        <a href="/about">О нас</a>
    </nav>
    <main>
        <h1>Привет, мир!</h1>
        <p>Это пример страницы на HTMLD.</p>
        <img src="dog.jpg" alt="Собака">
    </main>
    <footer>© 2023 Гуля ТВ</footer>
</body>
</htmld>
```
