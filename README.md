# HTMLD
## Теги (Тег в <>, чтобы закрыть </>)
1. <b>!DOCTYPE htmld</b>: Обязательный тег для объявления типа документа.
2. <b>htmld</b>: Корневой элемент документа.
3. <b>head</b>: Содержит метаинформацию о странице.
4. <b>title</b>: Заголовок страницы.
5. <b>body</b>: Основное содержимое страницы.
6. <b>header</b>: Шапка страницы.
7. <b>nav</b>: Навигационное меню.
8. <b>main</b>: Основной контент страницы.
9. <b>footer</b>: Подвал страницы.
10. <b>h1-h6</b>: Заголовки.
11. <b>p</b>: Параграф.
12. <b>img</b>: Изображение.
13. <b>a</b>: Ссылка.
## Новые теги (Тег в <>, чтобы закрыть </>):
### цифра.5 - пример
1. <b>dog</b>: Тег для вставки изображения или анимации собаки (вдохновлено Гуля ТВ).
1.5. dog src="dog.gif" alt="Собака Гуля"
2. <b>woof</b>: Тег для добавления звука собачьего лая (автовоспроизведение или по клику).
2.5. woof src="bark.mp3" autoplay
3. <b>paw</b>: Тег для создания кнопки в форме собачьей лапы.
3.5. paw onclick="alert('Гав!')">Нажми на меня!</paw
4. <b>tail</b>: Тег для создания анимации "виляния хвостом" (CSS-анимация).
4.5. tail speed="2">Виляние хвостом</tail
