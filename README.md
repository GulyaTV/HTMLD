# HTMLD
## Теги
1. ```<!DOCTYPE htmld>```: Обязательный тег для объявления типа документа.
2. ```<htmld>```: Корневой элемент документа.
3. ```<head>```: Содержит метаинформацию о странице.
4. ```<title>```: Заголовок страницы.
5. ```<body>```: Основное содержимое страницы.
6. ```<header>```: Шапка страницы.
7. ```<nav>```: Навигационное меню.
8. ```<main>```: Основной контент страницы.
9. ```<footer>```: Подвал страницы.
10. ```<p>```: Параграф.
11. ```<img>```: Изображение.
12. ```<a>```: Ссылка.
## Новые теги (Тег в <>, чтобы закрыть </>):
### Заголовки
1. ```<dog>```: Главный заголовок (```аналог <h1>```).
2. ```<puppy>```: Заголовок второго уровня (```аналог <h2>```).
3. ```<bark>```: Заголовок третьего уровня (```аналог <h3>```).
4. ```<tail>```: Заголовок четвёртого уровня (```аналог <h4>```).
5. ```<paw>```: Заголовок пятого уровня (```аналог <h5>```).
6. ```<whisker>```: Заголовок шестого уровня (```аналог <h6>```).
#### Пример кода
```
<!DOCTYPE htmld>
<htmld>
<head>
    <title>Моя страница</title>
</head>
<body>
    <dog>Добро пожаловать на сайт Гуля ТВ!</dog>
    <puppy>Наши услуги</puppy>
    <bark>Дрессировка собак</bark>
    <tail>Груминг</tail>
    <paw>Ветеринарная помощь</paw>
    <whisker>Контакты</whisker>
</body>
</htmld>
```
