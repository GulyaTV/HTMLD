# HTMLD (Hypertext Markup Language Dog)

HTMLD — это упрощённый язык разметки, вдохновлённый собаками и созданный для удобства разработчиков. Файлы HTMLD имеют расширение `.htmld`.

## Основные теги

### Заголовки
- `<dog>`: Главный заголовок (аналог `<h1>`).
- `<puppy>`: Заголовок второго уровня (аналог `<h2>`).
- `<bark>`: Заголовок третьего уровня (аналог `<h3>`).
- `<tail>`: Заголовок четвёртого уровня (аналог `<h4>`).
- `<paw>`: Заголовок пятого уровня (аналог `<h5>`).
- `<whisker>`: Заголовок шестого уровня (аналог `<h6>`).

### Пример использования
```htmld
<dog>Добро пожаловать на сайт Гуля ТВ!</dog>
<puppy>Наши услуги</puppy>
<bark>Дрессировка собак</bark>
<tail>Груминг</tail>
<paw>Ветеринарная помощь</paw>
<whisker>Контакты</whisker>
